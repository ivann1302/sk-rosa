# Структура блога для сайта ROSA

## Описание проекта

Блог будет содержать статьи по различным тематикам, связанным с ремонтом и отделочными работами. Статьи помогут клиентам получить полезную информацию, советы и рекомендации от профессионалов.

## Структура файлов

### HTML страницы

```
src/pages/
├── blog.html                    # Главная страница блога со списком всех статей
└── blog/                        # Папка для отдельных статей
    ├── article-template.html    # Шаблон для создания новых статей
    └── [статьи будут добавляться сюда]
```

### Стили (SCSS)

```
src/styles/components/blog/
├── _blog.scss                   # Основные стили блога (импорт всех остальных)
├── _blog-header.scss            # Стили для заголовка и описания блога
├── _blog-filter.scss            # Стили для фильтра по категориям
├── _blog-grid.scss              # Стили для сетки статей
├── _blog-card.scss              # Стили для карточек статей
└── _blog-article.scss           # Стили для страницы отдельной статьи
```

### JavaScript

```
src/scripts/features/blog/
├── blog-filter.js               # Фильтрация статей по категориям
└── blog-search.js               # Поиск по статьям (опционально)
```

## Функциональность

### Главная страница блога (`blog.html`)

1. **Заголовок и описание**
   - H1 заголовок "Блог ROSA"
   - Краткое описание блога
   - Информация о тематиках статей

2. **Фильтр по категориям**
   - Кнопки/табы для фильтрации:
     - Все статьи
     - Ремонт и отделка
     - Материалы и технологии
     - Советы и рекомендации
     - Новости и обновления
   - Активное состояние выбранной категории

3. **Сетка статей**
   - Адаптивная сетка (2 колонки на десктопе, 1 на мобильных)
   - Карточки статей с:
     - Изображением (превью)
     - Заголовком
     - Кратким описанием
     - Категорией (тег)
     - Датой публикации
     - Временем чтения (опционально)
     - Ссылкой на полную статью

4. **Пагинация или "Загрузить еще"**
   - Кнопка для загрузки дополнительных статей
   - Или классическая пагинация

### Страница отдельной статьи

1. **Хлебные крошки**
   - Главная → Блог → Название статьи

2. **Заголовок статьи**
   - H1 с названием
   - Мета-информация: дата, категория, автор (опционально)

3. **Изображение статьи**
   - Главное изображение (hero image)

4. **Содержание статьи**
   - Структурированный текст с заголовками
   - Изображения внутри статьи
   - Списки, цитаты
   - Выделение важной информации

5. **Блок "Поделиться"**
   - Кнопки для шаринга в соцсети

6. **Похожие статьи**
   - Блок с рекомендациями других статей

7. **Форма обратной связи** (опционально)
   - Кнопка "Задать вопрос" или "Заказать консультацию"

## Категории статей

1. **Ремонт и отделка**
   - Статьи о видах ремонта
   - Этапы ремонта
   - Стили интерьера

2. **Материалы и технологии**
   - Обзоры материалов
   - Новые технологии
   - Сравнение материалов

3. **Советы и рекомендации**
   - Практические советы
   - Что нужно знать перед ремонтом
   - Типичные ошибки

4. **Новости и обновления**
   - Новости компании
   - Обновления услуг
   - Специальные предложения

## SEO оптимизация

### Для главной страницы блога:
- Meta title: "Блог о ремонте | ROSA - Полезные статьи и советы"
- Meta description: "Читайте полезные статьи о ремонте квартир, выборе материалов, технологиях отделки. Советы от профессионалов ROSA."
- JSON-LD разметка для Blog
- Open Graph теги

### Для отдельных статей:
- Уникальный title для каждой статьи
- Уникальный meta description
- JSON-LD разметка для Article
- Open Graph теги с изображением статьи
- Canonical URL

## Интеграция с существующим сайтом

### Меню навигации
- Добавить пункт "Блог" в главное меню (после "Портфолио")
- Добавить ссылку в футер в раздел "Информация"

### Обновление файлов
- `src/index.html` - добавить ссылку в меню
- `src/pages/*.html` - добавить ссылку в меню на всех страницах
- `src/styles/main.scss` - добавить импорты стилей блога
- `src/sitemap.xml` - добавить страницы блога

## Дизайн

### Стиль карточек статей
- Похож на карточки портфолио, но адаптирован для статей
- Изображение сверху
- Заголовок
- Краткое описание
- Тег категории
- Дата публикации
- Hover эффект с легким подъемом карточки

### Цветовая схема
- Использовать существующие цвета из `_variables.scss`
- Акцентный цвет для категорий: `--color-green`
- Фон карточек: белый
- Тени: легкие, как в портфолио

### Адаптивность
- Десктоп: 2 колонки
- Планшет: 2 колонки (меньше отступы)
- Мобильный: 1 колонка
