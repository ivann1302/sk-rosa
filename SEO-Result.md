# SEO-аудит: Title, H1, Description

Анализ соответствия страниц критериям из SEO.md с учётом таблицы ключевых слов.

---

## Общие выводы

| Критерий | Статус |
|---|---|
| Ровно 1 H1 на каждой странице | ✅ Везде соблюдено |
| H1 уникальны внутри сайта | ✅ Соблюдено |
| Title уникальны | ✅ Соблюдено |
| Бренд в title через разделитель | ✅ Соблюдено |
| Гео в title и H1 | ⚠️ Нарушение на 2 страницах |
| Ключи кластера в title/H1 | ⚠️ Коммерческие хвосты пропущены |
| Каждый кластер — отдельная страница | ❌ 3 кластера не закрыты страницами |
| Длина description ~150–160 символов | ⚠️ Нарушение на 3 страницах |

## Страницы услуг

### 1. Главная — `index.html`

**Текущее состояние:**
- **Title:** `ROSA - профессиональный ремонт под ключ в Москве и Подмосковье`
- **H1:** `Профессиональный ремонт под ключ`
- **Description (182 симв.):** Ремонт квартир и домов под ключ в Москве и Московской области. Более 6 лет опыта, официальный договор, гарантия качества. Бесплатный замер и консультация. Звоните: +7 (985) 135-49-91

**Проблемы:**
- ❌ Title начинается с бренда `ROSA`, а не с ключевого слова. По SEO.md: _«Самые важные слова ставим в начало title»_.
- ❌ H1 не содержит гео (`в Москве`). По SEO.md: _«Для регионального сайта гео нужно добавлять в title и H1»_.
- ⚠️ В title нет слова «квартир» — а это один из главных ключей кластера.

**Рекомендации:**
```
Title:  Ремонт квартир под ключ в Москве и МО | ROSA
H1:     Ремонт квартир и домов под ключ в Москве
Desc:   Комплексный ремонт квартир и домов под ключ в Москве и МО. 6+ лет опыта,
        официальный договор, фиксированные цены. Бесплатный замер. ☎ +7 (985) 135-49-91
```

## Городские страницы (генератор)

Генератор работает через замену плейсхолдера города в шаблоне (`balashikha` → нужный город). Title/H1/Description для городских страниц унаследуют все проблемы шаблонных страниц (plastering, floor-screed, airless-painting, turnkey-repair) описанные выше. При исправлении основных страниц-шаблонов нужно пересгенерировать городские страницы командой `npm run generate`.

## Отсутствующие страницы (кластеры без посадочных)

По принципу из SEO.md: _«каждый кластер запросов — отдельная страница со своим title и H1»_. Следующие кластеры из таблицы **не имеют своей страницы**:

| Ремонт под сдачу | `ремонт под сдачу квартиры` | `/pages/repair-for-rent.html` |
| Ремонт помещений | `ремонт помещений`, `ремонт помещений под ключ`, `ремонт помещений под компьютерный клуб` | `/pages/commercial-repair.html` |
| Гипсовая штукатурка | `гипсовая штукатурка` | может быть раздел на plastering.html или отдельная страница |
| Цементная штукатурка | `цементная штукатурка` | аналогично |
| Штукатурка с маяками | `штукатурка с маяками` | аналогично |
