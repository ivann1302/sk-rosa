## robots.txt

**Текущее содержимое:**
```
User-agent: *
Allow: /
Disallow: /scripts/api/
Disallow: /test-*.php

# Страница 404 доступна для индексации
Allow: /404.html

Sitemap: https://sk-rosa.ru/sitemap.xml
```

### Проблемы

| # | Проблема | Приоритет |
|---|---|---|
| 1 | `Allow: /404.html` — бессмысленная и вредная строка | 🟡 Средний |
| 2 | `Disallow: /test-*.php` — шаблон с `*` в середине работает только в Google | 🟢 Низкий |

**Проблема 1 подробно:** `Allow: /` уже открывает весь сайт. Отдельный `Allow: /404.html` ничего не добавляет. Хуже того, комментарий говорит «доступна для индексации» — это неверная цель. Страница 404 возвращает HTTP 404, поэтому Google её не индексирует сам по себе. Если же сервер отдаёт 404-страницу с кодом 200 (soft 404), то явный Allow только усугубляет проблему.

**Проблема 2 подробно:** Стандарт robots.txt использует только **префиксное совпадение**. Шаблон `/test-*.php` с `*` в середине — расширение Google. Yandex может его не поддерживать. Надёжнее написать `Disallow: /test-` (если все тест-файлы имеют этот префикс).

### Рекомендуемый robots.txt

```
User-agent: *
Allow: /
Disallow: /scripts/api/
Disallow: /test-

Sitemap: https://sk-rosa.ru/sitemap.xml
```

### Проблема 4: Устаревшие даты lastmod 🟢

- Большинство страниц услуг: `lastmod=2025-01-20` — более года назад, хотя страницы явно обновлялись (git показывает изменения).
- Городские страницы: `lastmod=2026-02-03` — 3 недели назад, хотя git status показывает их как изменённые сейчас.

Поисковики ориентируются на lastmod при расстановке приоритетов краулинга. Устаревшие даты замедляют переобход.

## Итог

**robots.txt** — в целом правильный, нужно убрать 2 строки.

**sitemap.xml** — серьёзная проблема: **~128 реальных страниц сайта не попадают в индекс** из-за отсутствия в ситемапе. Это включает страницы по Москве, Жуковскому, Коломне и другим крупным городам. Одновременно ситемап ссылается на 20 несуществующих страниц, что снижает доверие поисковиков к файлу.

**Рекомендуется обновить ситемап вручную** или добавить автогенерацию ситемапа в скрипт генератора городских страниц.
